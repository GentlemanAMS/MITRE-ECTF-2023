fn send(&mut self, src: &mut [u8]) -> Result<(), CommunicationError>

Sends the data from src through the channel. Upon an error, a CommunicationError is given.

ERRORS:
  • CommunicationError::SendError
    • This could occur if any implementation-based error occurs while sending data.
  • CommunicationError::InternalError
    • This can occur if some internal error happens. This should only occur if something is wrong with the implementation.
