Module ucsc_ectf_util_common::messages

source ·
Expand description

This module is responsible for providing serde serializable/deserializable structs for messages sent between the car, key fob, and host tools.

Re-exports

pub use heapless;

Structs

DiffieHellmanMessage
A message containing the public key associated with the ephemeral secret and a key-signing public key to verify the public key’s authenticity. The public key associated with the ephemeral secret is signed by the private key paired with the key-signing public key.
EnableFeatureMessage
A message containing a signed packaged feature to enable a feature on a car. It is sent to a paired key fob associated with a car.
HostToolAck
An acknowledgement to a request sent by a host tool. The contained boolean is true if the requested operation was a success.
PackagedFeatureSigned
A signed packaged feature associated with the car it’s tied to. The signature guarantees that it’s not tampered with.
PackagedFeatureUnsigned
A packaged feature, containing the Car ID and Feature Number.
PairingChallenge
The message to send to a paired key fob that initiated a pairing sequence using a PairingRequest. The challenge is to prevent replay attacks.
PairingChallengeResponse
The response to send for a PairingChallenge. It contains the Nonce from the challenge to prevent replay attacks. See the fields of this struct for more information.
PairingPin
A struct containing the pairing pin needed to initiate a pairing sequence.
PairingRequest
A message to send to an unpaired key fob to initiate a pairing request containing a Nonce.
UnlockChallenge
The message to send to a paired key fob that initiated an unlock sequence using an UnlockRequest. This message contains a Nonce to prevent replay attacks.
UnlockChallengeResponse
The response to send for an UnlockChallenge. It contains the Nonce from the challenge to prevent replay attacks. See the fields of this struct for more information.
UnlockMessage
The message containing the unlock secret, the feature secrets of any enabled features on the car, and the car ID to be sent to the unlock host tool on a successful unlock.
UnlockRequest
The message to send to a car to signal the start of an unlock seequence. It contains the car ID of the car to be unlocked.
VerifiedPublicKey
A signed public key.
