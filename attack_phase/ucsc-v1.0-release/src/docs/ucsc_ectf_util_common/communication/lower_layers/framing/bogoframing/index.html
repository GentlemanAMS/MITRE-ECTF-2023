The functions in this module are to help implement BogoFraming based FramedTxChannels and RxChannels.

  • BogoFraming
    • BogoFraming is a very simple framing protocol. Each message begins and ends with one \1 character.
    • To prevent conflating \1 characters with the underlying data, the underlying data is hex encoded and decoded. NULL characters are completely ignored and wonâ€™t affect the message.
    • Helper functions to implement channels using this type of framing are in the bogoframing module.
